Raspberry Pi Setup for Development

Setting up SSH, I2C, SPI, Python, etc. on the Raspberry Pi on first time use

Required Components

  • Raspberry Pi

Steps

First time setup. use sudo raspi-config to set up time zone, enable I2C, SPI, SSH, and wifi.

Follow the prompts in order to set up the raspberry pi keyboard layout, username, and password.

Sign in, then Set up WiFi by going to System Options > Wireless LAN 

sudo raspi-config

In raspi-config, go to Interface Options, and enable SSH, SPI, and I2C. Then reboot. After logging back in, upgrade the OS: 

sudo apt-get update
sudo apt-get upgrade

If that fails, then run the following: 

sudo apt-get install -f
sudo reboot
sudo apt-get update
sudo apt-get upgrade
sudo reboot

Install python-related libraries 


sudo apt-get install python3-pip
sudo pip3 install --upgrade setuptools

Install adafruit libraries for development 

cd ~
sudo pip3 install --upgrade adafruit-python-shell
wget https://raw.githubusercontent.com/adafruit/Raspberry-Pi-Installer-Scripts/master/raspi-blinka.py
sudo python3 raspi-blinka.py
